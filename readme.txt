Get the metadata list from your WooCommerce order

== Description ==

English Bellow.

Este plugin é útil para obter informações de debug de plugins. Geralmente recomendando para quem tem [meus plugins](https://fernandoacosta.net/meus-plugins).

=== ## ===

Use this plugin to debug information about which meta is being save for each order on your WooCommerce store.

This plugin is useful for debug and to understand how the data is stored on database.

== Installation ==



== Frequently Asked Questions ==

= How can I see the metadata list? =

Navigate through WooCommerce -> Status -> Status -> Order Metadata and insert the order ID.
